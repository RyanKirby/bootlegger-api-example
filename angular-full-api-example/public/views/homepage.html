Welcome to the Bootlegger API


What is this?

This is an example/tutorial web app using the Bootlegger API. It is a fully working example created with Angular.js. It is a great place to start if you are trying to get familiar with the Bootlegger API. It uses every Bootlegger API endpoint allowing you to directly copy code and use it in your own project.

API Info!

Whenever you see a button click on it to find out details about what API Endpoint is being used and some information on what it does. It will also point you to the controller.js file the code is being used in so you can copy code to use in your own app!

Lets get started!

Firstly, lets kick things off by logging in! Click on 'Sign In' on the top bar above!

Note: If you are on a browser with console access (such as Chrome or FireFox), open the console to see the API Endpoints been called and the response from the Bootlegger API.
