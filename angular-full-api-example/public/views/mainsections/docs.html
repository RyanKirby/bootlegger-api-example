Documentation

This section will help you to navigate through the site and help you find a particular Api Endpoint. It also give a quick overview of the function of each API Endpoint.

Whenever you see a button click on it to find out details about what API Endpoint is being used and some information on what it does. It will also point you to the controller.js file the code is being used in so you can copy code to use in your own app!

Also rember to open the console if you can to see all server calls and the API's responses.

Visit https://bootlegger.tv/api/docs for more info.

Authentication:

Well well, you clever clogs! If you have gotten this far, you have already used an API Endpoint. Well done you! There are two main Authentication API Endpoints, /api/auth/login and /api/auth/logout. These are used in the NavBar fixed to the top of the screen

To see an example of these working go to index.js and public/ng/controllers/client_controller.js.

Profile:

get /api/profile/mine

Tested - This is used in the Shoots tab to list all the users shoots.

get /api/profile/me

Tested - This retrieves data about the user. Example can be found in the Top Bar.

get /api/profile/contributed

Tested - List all the shoots the user has contributed to. Example can be found on the Shoots page, in the 'Contributed' tab.

To see an example of these working go to public/ng/controllers/client_controller.js.
